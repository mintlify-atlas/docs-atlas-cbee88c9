## Overview

pInk provides powerful filtering capabilities through the `ControlsBar` component, allowing users to narrow down comics by multiple criteria simultaneously.

## Filter Types

The system supports three main filter categories:

  </Card>
  <Card title="Year" icon="calendar">

  </Card>

## ControlsBar Component

The filtering interface is implemented in `src/components/ControlsBar.tsx` and provides:

- **Filter dropdown** with checkbox selections
- **Active filter badge** showing count
- **Clear all filters** button
- **Dynamic filter options** based on available data

### Filter State Management

Filters are managed via React state passed from parent components:

```tsx Home.tsx
const [activeFilters, setActiveFilters] = useState<{
  publisher: string[];
  year: string[];
  language: string[];
}>({
  publisher: [],
  year: [],
  language: [],
});
```

### Filter Generation

Filter options are dynamically generated from the current dataset using `useMemo` (src/components/ControlsBar.tsx:59):

```tsx ControlsBar.tsx
const filters = useMemo(() => {
  const publishers = new Set<string>();
  const years = new Set<string>();
  const languages = new Set<string>();

  items.forEach((item) => {
    if ("publisher" in item && item.publisher) publishers.add(item.publisher);
    if (item.year) years.add(item.year.toString());
    if (item.language) languages.add(item.language);
  });

  return {
    publishers: Array.from(publishers).sort(),
    years: Array.from(years).sort((a, b) => parseInt(b) - parseInt(a)),
    languages: Array.from(languages).sort(),
  };
}, [items]);
```

<Note>
  Years are sorted in descending order (newest first) while publishers and languages are sorted alphabetically.
</Note>

## Filter Toggle Logic

Toggling a filter adds or removes it from the active filters array (src/components/ControlsBar.tsx:77):

```tsx ControlsBar.tsx
const toggleFilter = (type: "publisher" | "year" | "language", value: string) => {
  const current = activeFilters[type] || [];
  const updated = current.includes(value)
    ? current.filter((v: string) => v !== value)
    : [...current, value];

  setActiveFilters({ ...activeFilters, [type]: updated });
};
```

## Filter Application

Filters are applied in the parent component using `useMemo` for optimal performance (src/pages/Home.tsx:49):

```tsx Home.tsx
const filteredItems = useMemo(() => {
  let filtered = allComics.filter((item) =>
    item.title.toLowerCase().includes(searchTerm.toLowerCase())
  );

  // Apply year filter
  if (activeFilters.year.length > 0) {
    filtered = filtered.filter((item) =>
      activeFilters.year.includes(item.year?.toString())
    );
  }

  // Apply publisher filter
  if (activeFilters.publisher.length > 0) {
    filtered = filtered.filter((c) =>
      activeFilters.publisher.includes(c.publisher)
    );
  }

  // Apply language filter
  if (activeFilters.language.length > 0) {
    filtered = filtered.filter((c) =>
      activeFilters.language.includes(c.language)
    );
  }

  return filtered;
}, [allComics, searchTerm, activeFilters]);
```

## Click-Outside Handling

The filter dropdown automatically closes when clicking outside (src/components/ControlsBar.tsx:40):

```tsx ControlsBar.tsx
useEffect(() => {
  const handleClickOutside = (event: MouseEvent) => {
    if (
      filterContainerRef.current &&
      !filterContainerRef.current.contains(event.target as Node)
    ) {
      setIsFilterOpen(false);
    }
  };

  if (isFilterOpen) {
    document.addEventListener("mousedown", handleClickOutside);
  }

  return () => {
    document.removeEventListener("mousedown", handleClickOutside);
  };
}, [isFilterOpen]);
```

## Empty Results

When filters produce no results, the system displays an empty search state:

```tsx Home.tsx

  type={
    searchTerm || 
    activeFilters.year.length > 0 || 
    activeFilters.publisher.length > 0 || 
    activeFilters.language.length > 0 
      ? "empty_search" 
      : "empty_content"
  }
/>
```
