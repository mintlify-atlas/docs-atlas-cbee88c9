## What is pInk?

pInk is a high-performance SaaS comic book catalog system designed to provide a clean and intuitive interface for browsing, searching, and discovering your favorite comic books. Built with modern web technologies, pInk delivers a fast and responsive experience for comic enthusiasts.

## Why pInk?

pInk combines cutting-edge web technologies to deliver a superior comic browsing experience:

- **Lightning Fast**: Built with Bun and ElysiaJS for exceptional performance
- **Modern Stack**: React frontend with Vite for instant hot module replacement
- **Scalable Database**: Powered by Supabase (PostgreSQL) for reliable data storage
- **Type-Safe**: Full TypeScript support throughout the stack
- **Production Ready**: Optimized for deployment on Vercel and other platforms

## Technology Stack

pInk is built on a carefully selected stack of modern technologies:

| Component | Technology |
| --- | --- |
| Frontend | React 19 + Vite |
| Backend | Bun + ElysiaJS |
| Database | Supabase (PostgreSQL) |
| Language | TypeScript |
| Routing | React Router v7 |
| API Docs | Swagger (dev mode) |
| CORS | @elysiajs/cors |
