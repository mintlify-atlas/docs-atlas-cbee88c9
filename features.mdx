## Overview

pInk provides a comprehensive set of features designed to make browsing, discovering, and managing your comic book collection effortless and enjoyable.

## Core Features

### RESTful API

Access your comic catalog programmatically with a clean REST API:

```bash
# Get all comics
curl http://localhost:3000/api/comics

# Get specific comic by ID or slug
curl http://localhost:3000/api/comics/watchmen-1986

# Get all issues for a comic
curl http://localhost:3000/api/comics/1/issues

# Search and paginate issues
curl "http://localhost:3000/api/issues?limit=50&offset=0&search=batman"
```

<Note>
  In development mode, access interactive API documentation at `/swagger`
</Note>

## Data Features

### Comprehensive Metadata

Each comic and issue includes rich metadata:

<AccordionGroup>
  <Accordion title="Comic Metadata">

    - **Publisher**: Publishing house (Marvel, DC, etc.)
    - **Year**: Publication year
    - **Total Issues**: Number of issues in series
    - **Cover**: High-quality cover image URL
    - **Language**: Publication language

  </Accordion>

  <Accordion title="Issue Metadata">
    - **Title**: Individual issue title

## Developer Features

### API Documentation

Automatic Swagger documentation in development:

```typescript
import { swagger } from "@elysiajs/swagger";

const app = new Elysia()
  .use((app) =>
    process.env.NODE_ENV === "development" 
      ? app.use(swagger()) 
      : app
  );
```

Access at: `http://localhost:3000/swagger`
